## 🚀 Projetos em Destaque

### 🤖 JARVISKT
Assistente cognitivo experimental desenvolvido em **Kotlin + Jetpack Compose**, inspirado em interfaces de IA estilo terminal retrô.

Características:

- Motor de comandos
- Memória persistente em JSON
- Interface estilo terminal DOS
- Boot screen simulando inicialização de sistema

Explora conceitos como:

- interpretadores de comandos
- arquitetura modular
- persistência de dados

---

### 🏪 Mercora ERP
Sistema de gestão de mercado desenvolvido em **Kotlin + Compose Desktop**.

Objetivo do projeto:

Construir um sistema de mercado real com:

- cadastro de produtos

- estrutura básica de ERP
